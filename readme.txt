== Description ==

This plugin will save contact form 7 sumbition into database

Data can view through backend. Contact -> submitted data

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Contact-> screen to configure the plugin

== Frequently Asked Questions ==

= How to use the plugin? =
After activating the plugin go to "Contact -> Submitted Forms". You can see a drop down with submitted form names that created in CF7. Select the form that you want to view.
